Notes for Running MapperXML Library Example

NOTE: This example works, but is not yet complete. Additional features will be added in subsequent releases.

These notes are designed for testing with Tomcat 3 or 4. The examples also include an Ant build script. To run these examples you need the following components installed and configured:

  • Java VM 1.3.1
  • Tomcat 3.x or 4.x
  • MapperXML 1.9.2
  • Xerces 1.4.1
  • Oracle Database and JDBC Drivers (classes12.zip/jar)
  • Ant 1.4.1 (optional - needed to run build.xml script)

The following are the quick directions to install the examples:

Tomcat 3.x

  • Copy xerces.jar to [tomcat-install-dir]/lib/apps
  • Copy mapperxml.jar to [tomcat-install-dir]/lib/apps
  • Copy Oracle JDBC drivers, classes12.zip to [tomcat-install-dir]/lib/apps
  • Copy mapperlibrary.war to [tomcat-install-dir]/webapps
  • Start Tomcat
  • Browse to http://localhost:8080/mapperlibrary/library

Tomcat 4.x

  • Copy xerces.jar to [tomcat-install-dir]/lib
  • Copy mapperxml.jar to [tomcat-install-dir]/lib
  • Copy Oracle JDBC drivers, classes12.zip to [tomcat-install-dir]/lib
  • Copy mapperlibrary.war to [tomcat-install-dir]/webapps
  • Start Tomcat (Catalina)
  • Browse to http://localhost:8080/mapperlibrary/library

Database Creation/Population

You use the instructions in dbdesign.html to create and populate the database for the example.
