Shortcut Keys

The meta META key differs between operating systems. In general:
  • Windows/Linux: CTRL key.
  • Osx: The APPLE key.
Shortcut Key Action
META-o Open a file (password database).
META-s Save the current file (password database).
META-A Save as ... save the current database using another name.
META-N Create a new database.
META-Q Exit the application.
META-U Copy the user id (UID) to the clipboard.
META-P Copy the password (PWD) to the clipboard.
META-D Clear the contents of the clipboard, remove sensitive information.
META-E Make a new password entry.
ENTER Edit the currently selected password entry.
DELETE Delete the currently selected password id.
F1 Context sensitive help.
