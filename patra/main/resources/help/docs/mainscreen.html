Main Screen

Elements of the main screen

  1. Table view. This is a list of the password entries.
  2. The filter box. You can quickly search for a password entry by typing some text in this box. The table will be adjusted dynamically and you can quickly find an entry.
  3. The window title contains some useful information. It contains the full path to the database, and it also contains an indication if the database has changed since the last save operation.
  4. The main menu gives you access to all the Password Tracker functionality.
  5. The quick buttons are one of the ways to quickly copy the uid/password information to the clipboard. Take a look at the "using the clipboard" entry for more information on all possibilities.
