Using the clipboard

There are essentially 4 ways to copy a user id or a password to the clipboard. The clipboard will be cleared automatically when the application exits or you can do this manually using the menu bar or the context menu.

  1. Use the shortcut keys. The shortcut keys are mentioned in the menu bar.
  2. Use the quick buttons (left and right side, see preferences).
  3. Use the main menu bar.
  4. Use the context menu (right mouse button).
