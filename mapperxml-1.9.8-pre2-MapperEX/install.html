Notes for Running MapperXML Examples

These notes are designed for testing with any web application (war) server. The examples include an Ant build script. To run these examples you need the following components installed and configured:

  • Java VM 1.4
  • Web Application Server (Tomcat 4+, JBoss, etc)
  • Ant (optional - needed to run build.xml script)

The following are the quick directions to install the examples:

Tomcat

  • Copy mapperex.war to [tomcat-install-dir]/webapps
  • Start Tomcat
  • Browse to http://localhost:8080/mapperex

JBoss

  • Copy mapperex.war to [jboss-install-dir]/server/default/deploy
  • Start JBoss
  • Browse to http://localhost:8080/mapperex
