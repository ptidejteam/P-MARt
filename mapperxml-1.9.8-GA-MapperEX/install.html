Notes for Running MapperXML Examples

These notes are designed for testing with any web application (war) server. The examples include an Ant build script. To run these examples you need the following components installed and configured:

  • Java VM 1.4+
  • Web Application Server (Tomcat 4+, JBoss, etc)
  • Ant (optional - needed to run build.xml script)

Running Pre-Built Example

The following are the quick directions to deploy and run the pre-built examples:

  • Copy the file mapperex.war to your application server's deployment directory (or use it's deployment tool). For JBoss or Tomcat the location is: 
                   [jboss-install-dir]/server/default/deploy
               -- or --
               [tomcat-install-dir]/webapps
  • Start Tomcat or JBoss
  • Browse to http://localhost:8080/mapperex

Building and Running Example

The following are the quick directions to build and run the examples:

  • Create a ".build.properties" file in the root of the project
  • Add a property setting in the file: 
        deploy.dir=[jboss-install-dir]/server/default/deploy
               -- or --
               [tomcat-install-dir]/webapps
    where the deploy.dir is the path to your application server's deployment directory.
  • Run the Ant target "deploy" in build.xml
  • Start Tomcat or JBoss
  • Browse to http://localhost:8080/mapperlibrary
