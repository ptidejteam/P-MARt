Requirements

  • Java VM 1.4
  • Html/Xml parser: JTidy, Xerces or JAXP
  • For CodeGen tool: Velocity (with dependencies)
  • NOTE: The full distribution of Mapper includes JTidy, Xerces, and Velocity. They can be found in the lib directory.

General Installation

  • Download mapperxml-x.x.x.zip and unzip into a directory such as: /usr/local
  • Include mapperxml.jar in your project classpath

Deployment

  • Include mapperxml.jar in your deployment WAR file as a library
  • You may also need to include an xml parser (JTidy, Xerces, JAXP)
