Requirements

  • Java VM 1.3.1 (not yet tested with 1.4)
  • For examples: Xerces 1.4.4
  • For CodeGen tool: Velocity 1.2 (with dependencies)
  • NOTE: The full distribution of Mapper includes Xerces 1.4.4 and Velocity 1.2. They can be found in the lib directory.

General Installation

  • Download mapper-1.x.x.zip and unzip into a directory such as: /usr/local
  • Include mapperxml.jar in CLASSPATH for applications (found in /yourinstallpath/mapper-1.x.x/lib)

Installation for use with JBuilder 3 through 5

  • Follow General Installation first
  • Copy (or symlink) mapperxml.jar to <jbuilder-install-dir>/lib/ext
  • Start JBuilder
  • Select Tools/Configure Libraries and add "MapperXML" using mapperxml.jar

Installation for use with Forte 3

  • Follow General Installation
  • Mount mapperxml.jar in projects as needed
  • May also need to mount Xerces 1.4.4 if used

Installation for use with Tomcat 3 or 4

  • Follow General Installation
  • Copy mapperxml.jar to Tomcat's shared library (Tomcat 3 - /lib/apps 4 - /lib)

    - or -

  • Include mapperxml.jar in your deployment WAR file as a library
  • You will also need to do the same for xerces.jar if you are using the same xml parser as the examples.
